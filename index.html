What is ADK-Linux?

KDE Plasma Desktop

ADK-Linux is a linux distro based on Arch-linux with KDE Plasma Desktop, in order to give you the best cutting edge experience.

Development Software

This distro comes with development programs like Sublime Text, Meld and Kompare.

Arch Linux Based

Built on Arch Linux, ADK-Linux follows the KISS principles, keeping the base system minimal yet flexible.

Plasma Desktop

ADK-Linux uses Plasma Desktop with a simple dark theme.
Based around Aritim-Dark desktop layout, or install any theme you wish.
